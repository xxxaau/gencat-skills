# Títols, Estructura i Format — Comunicació Clara Gencat

## Títols i encapçalaments

### Regles fonamentals
- Màxim **100 caràcters**
- **Inicial en majúscula, resta en minúscules** (no tot en majúscules, no estil títol anglosaxó)
- **Sense punt final**
- **Sense dos punts finals** (excepte si el títol és una pregunta amb resposta al cos)
- Descriu el contingut de la secció amb les **paraules del lector**, no del redactor
- Ha de respondre a la pregunta: "Aquí trobaré..."

### Exemples

| ❌ Evita | ✅ Usa |
|---------|--------|
| INFORMACIÓ GENERAL | Informació general |
| Procediment Per a la Sol·licitud | Com sol·licitar la prestació |
| Resolució | Quan rebreu la resposta |
| Documentació Requerida | Quina documentació necessiteu |
| Terminis i condicions | Quan podeu presentar la sol·licitud |
| Aspectes Generals i Marc Normatiu | En quin marc legal s'emmarca |

### Jerarquia d'encapçalaments

```
H1 — Títol de pàgina (un sol H1 per pàgina)
  H2 — Secció principal
    H3 — Subsecció
      H4 — Apartat menor (usa amb moderació)
```

- No saltis nivells (no vagis de H1 a H3 directament)
- No uses encapçalaments per a maquetació o estètica, sinó per a estructura

## Paràgrafs

### Regles
- Un sol tema per paràgraf
- Comença per **la idea principal** (la informació més important primer)
- Longitud recomanada: **3-5 frases**
- Evita paràgrafs d'una sola frase (massa fragmentat) i de 10+ frases (massa dens)
- Usa el **punt i seguit** per separar idees d'un mateix paràgraf
- Usa el **punt i a part** per canviar de tema

### Estructura recomanada per a pàgines d'informació

```
1. RESUM INICIAL (el més important en 2-3 frases)
   → Qui pot demanar-ho / Què és / Per a qui va dirigit

2. INFORMACIÓ DETALLADA (seccions amb títols H2)
   → Requisits
   → Com fer-ho
   → Documentació necessària
   → Terminis
   → Resolució

3. MÉS INFORMACIÓ / CONTACTE (opcional)
   → Preguntes freqüents
   → On dirigir-se
```

## Llistes

### Quan usar una llista
- 3 o més elements del mateix tipus
- Seqüència d'accions ordenades
- Requisits o condicions
- Avantatges o característiques

### No uses llista si
- Tens menys de 3 elements (integra'ls a la frase)
- Els elements formen una seqüència narrativa
- Cada element necessita explicació llarga (usa paràgrafs)

### Format de llistes

**Llista no ordenada** (punts) — quan l'ordre no importa:
```
Necessiteu:
- El DNI original i una fotocòpia
- El certificat d'empadronament
- La sol·licitud emplenada
```

**Llista ordenada** (números) — quan l'ordre és important:
```
Per completar el tràmit:
1. Empleneu el formulari en línia
2. Adjunteu la documentació requerida
3. Signeu electrònicament
4. Envieu la sol·licitud
```

### Regles gramaticals de les llistes
- Si tots els elements continuen una frase introductòria, acaben en punt i coma (excepte l'últim, que acaba en punt)
- Si cada element és una frase completa, acaba en punt
- Si els elements són molt curts (1-3 paraules), no cal puntuació
- No mezcles estils en una mateixa llista
- Primer lletra minúscula si continua la frase introductòria; majúscula si cada element és frase completa

## Tipografia i format visual

### Font recomanada per aplicacions digitals
**Open Sans** (família sans-serif)

### Mida mínima
- Digital: mai per sota de **9px** (recomanat 11px+)
- Cos de text: 16px com a base

### Ús del negreta (**bold**)
- Per destacar informació **clau** dins un paràgraf
- **Poc** — si ho uses massa, deixa d'efectuar; màxim 1-2 termes per paràgraf
- Per a termes importants la primera vegada que apareixen

### Ús de la cursiva (*italic*)
- Títols de publicacions, obres, lleis: *Estatut d'Autonomia de Catalunya*
- Termes en altra llengua: el concepte *accountability*
- No usar per a emfatitzar (usa negreta per a això)

### Tot en majúscules (ALL CAPS)
- **Evita** en textos llargs — es llegeixen un 13% més lent
- Acceptable en sigles: DNI, NIF, IRPF, SEPE
- No per a títols ni encapçalaments

### Subratllat
- **Reserva'l exclusivament per a hiperenllaços** en context digital
- Mai per emfatitzar — confondria amb un enllaç

### Alineació del text
- **Digital**: alineació **esquerra** (millor llegibilitat, especialment per a persones amb dislèxia)
- **Imprès**: justificat (tradicional, acceptable si s'eviten forats entre paraules)
- **Mai**: alineació dreta per a textos llargs

### Interlineat (espaiat entre línies)
- **Digital**: mínim **1,5** dins el paràgraf
- Espai entre paràgrafs: almenys 1,5 vegades l'interlineat

## Nombres i dates

### Dates
- Format estàndard: **DD/MM/AAAA** (en formularis i camps de dades)
- Format llegible: **15 de març de 2024** (en textos narratius)
- Mai el format anglosaxó: ~~03/15/2024~~

### Nombres
- Separador decimal: **coma** (14,5 — no 14.5)
- Separador de milers: **punt** (1.500.000 — no 1,500,000)
- Percentatges: **14,5%** (sense espai entre número i %)
- Moneda: **14,50 €** (espai entre número i símbol)

### Hores
- Format 24h en contextos digitals: **14:30**
- Format 12h si s'especifica: **2:30 del migdia** / **2:30 de la tarda**

## Sigles i acrònims

- **Primera vegada**: escriu el nom complet seguit de la sigla entre parèntesis
  - "El Document Nacional d'Identitat (DNI)..."
- **Les vegades següents**: usa la sigla sola (DNI)
- Si el text és curt, pots escriure sempre el nom complet
- Les sigles no porten punt entre les lletres: **DNI** (no D.N.I.)
- Les sigles no porten accent: **IVA** (no ÍVA), **URL** (no URĹ)

## Pàgines d'error: estructura recomanada

```
TÍTOL (H1): Text clar que descriu el problema en termes humans
  → "No hem trobat la pàgina" (no "Error 404")
  → "Hi ha hagut un problema" (no "Error 500 - Internal Server Error")

COS: Explicació breu + instrucció d'acció
  → Màxim 2-3 frases
  → Inclou què fer a continuació

ACCIONS (botons):
  → Acció primària: Tornar a l'inici / Tornar enrere
  → Acció secundària: Contactar suport (si és necessari)
```
